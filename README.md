EduFree is a full-stack web application developed to support a non-profit initiative that provides **free and quality education** to children in rural communities through volunteer-driven learning.

The platform enables volunteers to upload educational content, while admins review and approve the content to maintain quality before it is made available to learners.

---

## About EduFree

EduFree aims to bridge the educational gap in under-resourced areas by connecting passionate volunteers with students who lack access to structured learning resources.

Through online educational videos and admin moderation, EduFree ensures reliable and meaningful learning experiences for students.

---

## Key Features

### Admin
- Secure admin login
- Approve or reject uploaded educational videos
- Ensure content quality and moderation

### Volunteers
- Upload educational videos
- Add title and description to content
- Track upload status (Pending / Approved)

### Learners
- View only admin-approved educational videos
- Simple and easy-to-use interface
